<unordered_map>
Feedback
Summarize this article for me
Defines the container class templates
unordered_map
and
unordered_multimap
and their supporting templates.
Syntax
#include <unordered_map>
Note
The <unordered_map> library also uses the
#include <initializer_list>
statement.
Classes
Class
Description
unordered_map Class
Stores hash table of {key, mapped} pairs.

Stores hash table of {key, mapped} pairs.
